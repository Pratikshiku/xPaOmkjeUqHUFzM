## Student Graduation Departure System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/34d72e0e20bc41c290c3341b9ea5023c/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): graduates, teachers, administrators.

**Graduate Functions:**
1. Login
2. Registration
3. Homepage
4. Graduation Process
5. Website Announcements
6. Message Feedback
7. Personal Center
8. My Fee Settlement
9. My Thesis Review
10. My Favorites Management


**Teacher Functions:**
1. Login
2. Registration
3. Homepage
4. Graduation Process
5. Website Announcements
6. Message Feedback
7. Personal Center
8. Student Management
9. Graduation Process Management
10. Fee Settlement Management
11. Thesis Review Management



**Administrator Functions:**
1. Login
2. Personal Center
3. Student Management
4. Teacher Management
5. Graduation Process Management
6. Fee Settlement Management
7. Thesis Review Management
8. Message Board Management
9. Administrator Management
10. Carousel Management
11. Website Announcements
